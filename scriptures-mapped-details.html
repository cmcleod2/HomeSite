Weather Web App

This is a web app built using only Javascript and HTML that allows the user to browse the Bible and other scriptures. When the site displays the text of the scripture the site will automatically recenter the map and drop pins on the geotagged locations in that particular scripture. For example, if the user enters Genesis 37 the map will zoom out and drop pins on Egypt, Gilead, Canaan, Dohan, and Shechem. If you aren't sure where those places really are then this is the app for you! This project was built using an API provided by Dr. Stephen Liddle and the Google Maps API.
